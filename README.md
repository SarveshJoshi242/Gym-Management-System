🏋️ Gym Management System (GMS)

A web-based Gym Management System developed using Node.js with a local server, designed to efficiently manage gym operations 
through an interactive admin dashboard.

🚀 Features
📊 Admin Dashboard
  Overview of total members, attendance, classes, and inquiries
  Clean and responsive UI built using Tailwind CSS
   
👥 Member Management
  Add, view, and manage gym members
  Membership categorization (Basic, Premium, VIP)

📅 Attendance Tracking
  Daily attendance marking system
  Attendance statistics (present, absent, percentage)

🏋️ Class Management
  View and manage scheduled gym classes

📩 Inquiry Handling
  Track and respond to customer inquiries

🛠️ Tech Stack
Backend: Node.js (Local Server)
Frontend: HTML, Tailwind CSS, JavaScript
Icons & UI: Font Awesome + Google Fonts

🎯 Objective
The aim of this project is to digitize gym management processes and provide an easy-to-use dashboard for administrators to handle 
members, attendance, and daily operations efficiently.

💡 Key Highlights
Modular dashboard design with multiple functional tabs
Interactive UI with dynamic data rendering
Scalable structure for future backend/database integration

🚀 Future Improvements
Database integration (MongoDB)
Authentication system (Admin login)
Cloud deployment
AI-based fitness recommendations (planned)

▶️ How to Run
1. Clone the repository
2. Navigate to project folder
3. Install dependencies:
   ```
   npm install
   ```
4. Start server:
   ```
   node app.js
   ```
5. Open in browser:
   ```
   http://localhost:3000
   ```
